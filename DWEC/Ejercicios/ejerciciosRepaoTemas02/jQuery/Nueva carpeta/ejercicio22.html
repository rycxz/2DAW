HTML es el lenguaje que se emplea para el desarrollo de páginas de internet.

Este lenguaje está constituido de elementos que el navegador interpreta y las muestra en la pantalla de acuerdo a su contenido. Existen elementos para posicionar imágenes sobre una página, hipervínculos que nos permiten dirigirnos a otra página, listas, tablas, etc.

Para poder crear una página HTML solo necesitamos un simple editor de texto y un navegador de internet (IExplorer, FireFox etc.), emplearemos el navegador para mostrar el contenido (recorder que ahora estamos viendo una página HTML con su navegador).

Lo más importante es que en cada MOMENTO HAGAIS los ejercicios propuestos y modifiqueislos que se os dan ya resueltos.

Estas clases lo que buscan es acercar el lenguaje HTML + jQuery a un alumno que nunca antes trabajó con el mismo. No pretende mostrar todas los elementos HTML + jQuery en forma secuencial.

Como veremos, de cada concepto se presenta una parte teórica, en la que se da una breve explicación y luego se pasa a la sección del ejercicio resuelto donde podréis ver el contenido de la página HTML + jQuery y cómo la visualiza en vuestro navegador. Por ultimo, se os propone que hagais páginas en forma autónoma.
